# Contributors

LinguaMedica RW is both a software project and a linguistic research contribution.
This file documents **intellectual credit** — who created the translations,
neologisms, and etymological analyses that make up the dictionary data.

For code contributions, see the Git history. For dictionary data attribution,
the `contributed_by` and `source` fields on each term provide per-entry provenance.

## How Attribution Works

Each term in the database carries:
- `contributed_by` — the person who created or validated the translation
- `source` — the document or context the term was drawn from
- `date_added` — when the term entered the dictionary

The dictionary data is licensed under **CC BY 4.0** (see [LICENSE-DATA](LICENSE-DATA)).
If you reuse any translations, you must credit the contributor listed on that term.
